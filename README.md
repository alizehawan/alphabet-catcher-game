# Alphabet Catcher Game (Assembly)

This is a console-based **Alphabet Catcher Game** written in Assembly Language.

## Features
- **Single-player mode**: Catch falling alphabets using keyboard controls.
- **Two-player mode**: Compete with a friend on the same console.
- **Timer**: Game is time-limited; the player with the highest score wins.
- **Score tracking**: Catch the correct alphabet to increase your score.
- **Console-based graphics**: Simple text-based gameplay.

## Concepts Highlighted
- Assembly language programming (x86 or MASM syntax)
- Loops and conditional jumps (JMP, JE, JNE, etc.)
- Keyboard input and player movement
- Timer implementation using low-level instructions
- Memory management and variables
- Basic game logic at low-level programming

## Instructions
1. Assemble the program using an assembler like MASM or TASM.
2. Run the executable in a DOS or console environment.
3. Follow on-screen instructions for single or two-player mode.
4. Use keyboard controls to catch falling letters.
5. Try to get the highest score before the timer ends.
